# 題組定義

## 題組架構說明

- 題組是逐行解鎖模式下選擇練習範圍的基本單位
- 每個題組對應特定的字元集合
- 題組編號格式為 `1-序號`
- 平假名與片假名共用相同的題組編號系統，但屬於不同主線，進度完全獨立
