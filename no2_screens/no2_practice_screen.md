# 練習畫面

## 畫面目標

- **作為:** 練習主線與難度選擇的入口畫面
- **提供:** 練習主線與難度選擇，進入刷題畫面的入口

---

## 導航結構

- 主線由底部 Nav Bar 的 tab 決定，畫面本身不設主線切換
- `平假名` tab 顯示平假名練習，`片假名` tab 顯示片假名練習，結構完全相同

---

## 畫面佈局

四種難度各自以大卡片呈現，由上而下排列，卡片之間留有間距。

```
+---------------------------+
|  平假名                   |
+---------------------------+
|                           |
|  +---------------------+  |
|  |     逐行解鎖        |  |
|  +---------------------+  |
|                           |
|  +---------------------+  |
|  |    全字符混合       |  |
|  +---------------------+  |
|                           |
|  +---------------------+  |
|  |     聽選模式        |  |
|  +---------------------+  |
|                           |
|  +---------------------+  |
|  |      四選一         |  |
|  +---------------------+  |
|                           |
+---------------------------+
| [平假名][字元總覽][片假名]|
+---------------------------+
```

---

## 難度選項行為

**逐行解鎖**

- **觸發:** 點擊卡片
- **導航:** Segue 進入題組列表畫面，詳見題組列表畫面規格

**全字符混合**

- **觸發:** 點擊卡片
- **導航:** Modal 進入刷題畫面
- **題庫:** 當前主線全部字元

**聽選模式**

- **觸發:** 點擊卡片
- **導航:** Modal 進入刷題畫面
- **音頻模式:** 選項發音模式，字卡靜音

**四選一**

- **觸發:** 點擊卡片
- **導航:** Modal 進入刷題畫面
- **音頻模式:** 選項發音模式，字卡靜音
- **選項數量:** 四個
