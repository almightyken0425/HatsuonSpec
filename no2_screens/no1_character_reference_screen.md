# 字元總覽畫面

## 畫面目標

- **作為:** App 預設首頁
- **提供:** 平假名與片假名並排展示的完整字元表，讓使用者同時看到同一個音的兩種寫法

---

## 導航結構

- 平假名與片假名合併在同一頁顯示，無頂部切換 tab
- 底部固定顯示 App 導航列，包含 `字元總覽` 與 `練習` 兩個 tab

---

## 字元卡片格式

每張卡片上方顯示平假名，中間顯示片假名，底部顯示 romaji。同一個音的兩種寫法共用一張卡片。

```
+------+
|  あ  |
|  ア  |
|  a   |
+------+
```

長音符 ー 無平假名對應，卡片上方留空，僅顯示片假名與 romaji。

```
+------+
|      |
|  ー  |
|  ー  |
+------+
```

|  +----+                   |

|  | オ |                   |

---

## 字元網格規則

- **清音:** 以 5 欄網格排列，ヤ行與ワ行的空位顯示為空白格
- **拗音:** 以 3 欄網格排列
- **促音、長音符:** 獨立成段
- 字元卡片只顯示假名字形與 romaji，不顯示任何正確率或熟練度指標

---

## 字元卡片互動

- **點擊字卡:** 播放對應音的發音
- 平假名與片假名讀音相同，共用同一音檔
