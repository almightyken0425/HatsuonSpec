# 題組列表畫面

## 畫面目標

- **作為:** 逐行解鎖難度的題組選擇畫面
- **提供:** 當前主線全部 21 個題組的卡片式列表，顯示各題組的練習進度
- **支援:** 單選題組進入標準模式，或切換多選模式選取多個題組進入自由模式

---

## 畫面佈局

**題組卡片列表:**
- 四欄網格排列
- 依分段顯示，各段前方附段落標題
    - 清音
    - 濁音・半濁音
    - 拗音
    - 促音・長音符

**題組卡片:**
- 代表字元
- 進度填色區
    - 填色比例代表該題組的練習進度

## 進度填色計算

- **資料來源:** 進度追蹤模組，詳見 `no3_module_specs/no4_progress_module.md`
- 進度值僅反映標準模式的累積答題歷史，自由模式的答題不計入
- 平假名與片假名兩條主線各自獨立計算

## 互動行為

**預設狀態**

- **點擊卡片:**
    - **導航:** Modal 進入該題組的標準模式刷題
    - 答題進度跨 session 保存
- **點擊 `多選`:** 切換至多選模式

**多選模式**

- **點擊卡片:** 切換選中狀態，選中的卡片以高亮底色顯示
- **`開始自由練習` 啟用條件:** 至少選中一個題組
- **點擊 `開始自由練習`:**
    - **導航:** Modal 進入自由模式刷題
    - 本次 session 的答題紀錄不保存至歷史
- **點擊 `完成`:** 離開多選模式，回到預設狀態，選中狀態清除
